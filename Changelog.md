# Changelog

All notable changes to this project will be documented in this file.

The project was developed as a continuous implementation effort.
Formal semantic versioning begins from v0.1.0.

---

## [0.1.0] - 2026-02-14

### Added

- Initial full-stack task management application.
- Frontend built with React and TypeScript.
- Backend built with Node.js and Express.
- PostgreSQL database integration using Prisma ORM.
- JWT-based authentication (register, login, profile).
- Project management functionality (create, update, delete).
- Task CRUD operations within projects.
- Drag-and-drop task board interface.
- Real-time updates using Socket.io.
- Docker and docker-compose configuration.
- Basic API route structure and middleware.
- Initial project documentation (README).
